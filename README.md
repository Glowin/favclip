FavClip
=============

FavClip是从豆瓣网自动分析热门相册并同步保存到人人网账号的自动程序。

使用的类库
-------------
[PHP Simple HTML DOM Parser](http://simplehtmldom.sourceforge.net/)   
>A HTML DOM parser written in PHP5+ let you manipulate HTML in a very easy way!
